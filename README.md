# ✈️ 비행기 탑승 게이트 놀이 앱

아이들이 표(탑승권)를 카메라에 보여주면
- 맞는 표 → **"인증되었습니다"** (초록 화면 + 딩동 소리 + 음성)
- 6초 안에 표를 못 찾으면 → **"다시 인증해주십시오"** (빨간 화면 + 부저 소리 + 음성)

## 완성까지 남은 3단계

### 1. 표 그림 준비
탑승권으로 쓸 그림 1장 (사진 파일: jpg 또는 png)
- **알록달록하고 복잡한 그림일수록 잘 인식됩니다.** (단순한 단색 도형, 글자만 있는 그림은 인식이 잘 안 됨)
- 이 그림을 인쇄해서 아이들에게 나눠주면 그게 탑승권이 됩니다.

### 2. 인식 데이터(targets.mind) 만들기 — 1번만 하면 됨
1. 브라우저에서 열기: https://hiukim.github.io/mind-ar-js-doc/tools/compile
2. 표 그림 파일을 끌어다 놓기 → **Start** 버튼
3. 완료되면 **Download** → `targets.mind` 파일이 받아짐
4. 그 파일을 이 폴더(`index.html` 옆)에 넣기

### 3. 패드에서 열기
카메라는 보안 때문에 **인터넷 주소(https)로 열어야** 작동합니다.
- 가장 쉬운 방법: https://app.netlify.com/drop 에 이 폴더를 통째로 끌어다 놓으면 무료 주소가 생김
- 그 주소를 패드 브라우저(Chrome/Safari)로 열고, 카메라 권한 "허용"

## 놀이 방법
- 게이트 직원 역할이 패드를 들고 "게이트 열기"를 누름
- 아이가 표를 카메라(후면)에 보여주면 자동으로 "인증되었습니다"
- "탑승권 검사" 버튼을 누르면 6초 카운트다운 — 그 안에 표가 없으면 "다시 인증해주십시오"

※ 패드에 인터넷이 연결되어 있어야 합니다.
